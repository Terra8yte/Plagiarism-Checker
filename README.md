# Universal Plagiarism Checker

A full-stack plagiarism checker built using Java Spring Boot with support for text comparison, PDF/DOCX extraction, and web plagiarism detection.

## Features

* Text plagiarism checking
* PDF and DOCX support
* Web plagiarism detection
* Source matching
* Similarity percentage calculation

## Technologies Used

* Java
* Spring Boot
* Thymeleaf
* Apache PDFBox
* Apache POI
* JSoup
* Jackson
* Maven

## Run Project

```bash
mvn spring-boot:run
```

Open:

```text
http://localhost:8080
```
